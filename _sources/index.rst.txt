Pangeo / ESGF Cloud Data Working Group
======================================

`Pangeo <http://pangeo.io/>`_ and the `Earth System Grid Federation <https://esgf.llnl.gov/>`_
have established an ad-hoc working group to help coordinate efforts related to
storing and cataloging CMIP data in the cloud.

This website describes the activities of this working group and provides technical
documentation for accessing the cloud data.
