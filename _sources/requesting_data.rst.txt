Requesting CMIP6 data to be added to the cloud holdings
========================================================

If you want to request the addition of new CMIP6 data to either the netcdf or zarr buckets, please follow the steps below. 

For netcdf files: Submit a GitHub issue `here <https://github.com/pangeo-data/pangeo/issues>`_ 

For zarr files: See `this <https://github.com/leap-stc/cmip6-leap-feedstock#how-can-i-request-new-data>`_
